# Chat with AI

This guide will show you how to chat with AI using the `ai-chat` command.

## Integrate with LLM

To chat with AI, you need to integrate with LLM.  This will allow you to chat with AI using the `ai-chat` command.

## Voice Recognition

You can use the `voice` command to recognize the voice input from the user.

## Text to Speech

After receiving the response from AI, you can use the `tts` command to convert the text to speech.

## Emoji and Actions

Difference with a general chatbot, humanoid can use emoji and actions to emphasize the expression other than voice output. For example, if the response is suitable with happyiness emotions, then we should use happy emoji and happy actions to make the conversation more lively.
