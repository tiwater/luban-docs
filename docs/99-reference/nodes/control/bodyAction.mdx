# Body

**Description**: Control the movement of the robot's body

**Type**: robot

## Inputs

| Name | Type | Description |
| --- | --- | --- |
| **Squat** | number | Controls the up-and-down movement, ranging from -0.15 to 0. Negative values for downward motion, 0 for neutral position |
| **Rotate Waist** | number | Controls left-and-right rotation, with a range of -14.32 to 14.32. Positive values for left rotation, negative for right rotation. Precision of 8 decimal places |
