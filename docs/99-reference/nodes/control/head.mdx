# Head

**Description**: Control the movement of the robot's head

**Type**: robot

## Inputs

| Name | Type | Description |
| --- | --- | --- |
| **Roll** | number | Specify the rotation around the x-axis. Negative values turn the head to the left, and positive values turn it to the right. Range: -17.1887 to 17.1887 |
| **Pitch** | number | Specify the rotation around the y-axis. Positive values tilt the head forward, and negative values tilt it backward. Range: -17.1887 to 17.1887 |
| **Yaw** | number | Specify the rotation around the z-axis. Negative values twist the head to the left, and positive values twist it to the right. Range: -17.1887 to 17.1887 |
