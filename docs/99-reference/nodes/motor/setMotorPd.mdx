# Set Motor PD Parameters

**Description**: Set the parameters for a Proportional-Derivative (PD) control mode for a specific motor

**Type**: robot

## Inputs

| Name | Type | Description |
| --- | --- | --- |
| **No** | text | Motor number |
| **Orientation** | text | Motor orientation |
| **P** | number | Proportional gain value |
| **D** | number | Derivative gain value |

| --- | --- | --- |
