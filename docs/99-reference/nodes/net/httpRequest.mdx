# HTTP Request

**Description**: Perform an HTTP access request

**Type**: function

## Inputs

| Name | Type | Description |
| --- | --- | --- |
| **URL** | text | The URL to visit |
| **Headers** | text | Http headers, in json format |
| **Body** | text | Http request body |
## Outputs

| Name | Type | Description |
| --- | --- | --- |
| **Data** | text | Response of the Http request |
| **Status** | text | Http status code |
## Options

| Name | Type | Description |
| --- | --- | --- |
| **Timeout** | number | Timeout |
| **NodeLogic** | text | Node logic |
| **Method** | text | Http request method |
