# Load URL Resource

**Description**: Get the content specified by the URL

**Type**: function

## Inputs

| Name | Type | Description |
| --- | --- | --- |
| **URL** | text | The URL to be accessed. Only support GET method |
| **Headers** | text | Http Headers, in json format |
## Outputs

| Name | Type | Description |
| --- | --- | --- |
| **Data** | text | Content of the resource specified by the URL |
## Options

| Name | Type | Description |
| --- | --- | --- |
| **NodeLogic** | text | Node logic |
