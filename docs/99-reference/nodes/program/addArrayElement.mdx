# Add Array Element

**Description**: Add elements to the array. If you add more than one element at a time, keep the newly added element ID in element2, element3 form

**Type**: function

## Inputs

| Name | Type | Description |
| --- | --- | --- |
| **Input Array** | json | The original array |
| **Element 1** | any | The element to be added |
## Outputs

| Name | Type | Description |
| --- | --- | --- |
| **Output Array** | json | The array with the new added elements |
## Options

| Name | Type | Description |
| --- | --- | --- |
| **NodeLogic** | any | Node logic |
| **Index** | number | The index to insert the element at. -1 means at the end of the array. |
