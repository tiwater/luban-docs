# Get Array Element

**Description**: Get element from the array by index and remove it if required.

**Type**: function

## Inputs

| Name | Type | Description |
| --- | --- | --- |
| **Input Array** | json | The original array |
## Outputs

| Name | Type | Description |
| --- | --- | --- |
| **Element** | any | Retrieved element |
| **Output Array** | json | The array after retrieving and removing the item if specified |
## Options

| Name | Type | Description |
| --- | --- | --- |
| **Index** | number | Index of the item to retrieve. -1 means at the end of the array. |
| **Remove** | boolean | Whether to remove the retrieved item from the array |
| **NodeLogic** | any | Node logic |
