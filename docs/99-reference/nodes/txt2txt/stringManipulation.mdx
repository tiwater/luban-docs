# String Manipulation

**Description**: Replace the {{string}} format string in the template, where "string" the ID of the corresponding input port

**Type**: function

## Inputs

| Name | Type | Description |
| --- | --- | --- |
| **Template** | text | String template |
## Outputs

| Name | Type | Description |
| --- | --- | --- |
| **Result** | text | The processed string |
## Options

| Name | Type | Description |
| --- | --- | --- |
| **NodeLogic** | text | Node logic |
| **Escape** | boolean | Whether escape special characters such as double quotation marks and line breaks in strings |
