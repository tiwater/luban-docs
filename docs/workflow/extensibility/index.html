Extensibility

Workflow templates are a way to create a new workflow with a predefined structure. You can use them to create a new workflow with a specific structure and configuration.

Build from Template​

To create a new workflow from a template, follow these steps:

  1. Click the New Workflow button in the top right corner of the application.
  2. Select the template you want to use from the list of available templates.
  3. Click the Create button to create a new workflow with the selected template.

Publish Template​

If you have created a workflow that you think others might find useful, you can publish it as a template. To publish a workflow as a template, follow these steps:

  1. Open the workflow you want to publish as a template.
  2. Click the Publish as Template button in the top right corner of the application.
  3. Enter a name and description for the template.

After publishing the template, it will be available to other users when creating a new workflow.

Manage templates​

You can manage your templates from the Templates tab in the application. From here, you can view, edit, and delete your templates.

To view the list of available templates, click the Templates tab in the application. From here, you can see a list of all the templates available to you.

To edit a template, click the Edit button next to the template you want to edit. From here, you can make changes to the template and save your changes.

To delete a template, click the Delete button next to the template you want to delete. You will be prompted to confirm the deletion before the template is deleted.
